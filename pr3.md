# Практическая работа №3: "Объекты"

---

## Задание

Вам нужно создать файл `index.js` и выполнить 2 последовательных шага. 
**Важно:** Нельзя написать весь код сразу. Каждый шаг требует запуска кода и копирования результата.

### Шаг 1: Создание объекта
1. Создайте объект `student`.
2. Добавьте в него свойства:
   - `id`: должно генерироваться через `Date.now()`.
   - `name`: ваше имя (латиницей).
   - `prefix`: первые 3 буквы вашего имени в нижнем регистре (например, `nik` для Nikita).
   - `status`: строка `"active"`.
3. Добавьте метод `getStatus()`, который возвращает текущее значение `status`.
4. **Действие:** Запустите файл в консоли. Скопируйте весь вывод `console.log(student)`.
5. **В коде:** Создайте переменную `const SNAPSHOT_STEP1 = "..."` и вставьте туда скопированный вывод как строку.

### Шаг 2: Изменение состояния
1. Добавьте в объект `student` метод `deactivate()`.
2. Этот метод должен менять свойство `status` на `"inactive"`.
3. Вызовите этот метод в коде.
4. **Действие:** Снова запустите файл. Скопируйте новый вывод `console.log(student)`.
5. **В коде:** Создайте переменную `const SNAPSHOT_STEP2 = "..."` и вставьте туда новый вывод.
